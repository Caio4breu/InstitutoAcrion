# InstitutoAcrion
Desenvolvimento colaborativo de um website para a empresa fictícia "Instituto Acrion", cuja foco e objetivo tem o aluguel de campos esportivos.

## Origem do Projeto
O desenvolvimento teve inicio em uma atividade da aula de Engenharia de Requisitos dada pelo professor Silvio.

## Desafio
O Instituto Acrion precisa de uma plataforma para administrar seu negocio. A empresa gerencia oito (8) espaços esportivos que são alugados por diversos tipos de clientes, porem o metodo usado até o começo desse projeto era uma agenda atualizada esporadicamente de maneira manual sem nenhum padrão pré estabelecido.

## Solução
A plataforma deve fornecer métodos de organização onde os clientes possam escolher entre datas disponiveis e os administradores consigam gerenciar com maior eficiencia e praticidade.
