# **InstitutoAcrion**
Desenvolvimento colaborativo de um website para a empresa fictícia "Instituto Acrion", cujo foco e objetivo é o aluguel de campos esportivos.

## **Origem do Projeto**
O desenvolvimento teve início em uma atividade da aula de Engenharia de Requisitos dada pelo professor Silvio.

### **Desafio**
O Instituto Acrion precisa de uma plataforma para administrar seu negócio. A empresa gerencia oito (8) espaços esportivos que são alugados por diversos tipos de clientes, porém o metodo usado até o começo desse projeto era uma agenda atualizada esporadicamente de maneira manual sem nenhum padrão pré estabelecido.

### **Solução**
A plataforma deve fornecer métodos de organização onde os clientes possam escolher entre datas disponiveis e os administradores consigam gerenciar com maior eficiencia e praticidade.

### **Público Alvo**
#### *Corporativo*
Empresas, firmas e corporações geralmente possuem atividades de lazer para recompensar os esforços de seus funcionários ou para elevar a moral da equipe. Um jogo de futebol é perfeito para criar um sentimento de união e camaradagem além de reforçar uma competição saudavel entre os participantes.

#### *Escolar*
Muitas escolas possuem jogos internos, eventos anuais onde todas as turmas competem entre si, porém muitas dessas escolas não possuem a infra estrutura para poder suprir as necessidades do evento. O Instituto Acrion possui espaço o suficiente para pretigiar um evento desse porte. 

#### *Prefeitura*
Esse tipo de evento geralmente é mais comum em escolas particulares, portanto existe a possibilidade de firmar um acordo com a prefeitura para que escolas públicas possam realizar tais eventos dentro do complexo esportivo.

#### *Eventos Particulares*
Goiânia é cheia de indivíduos com muito dinheiro e tempo disponivel. Abrir o espaço para eventos particulares onde grupos de amigos possam reservar um campo de futebol para uso próprio é extremamente possível, podendo ainda desenvolver uma comunidade de clientes fiéis que podem interagir com frequência nas redes sociais

#### *Jogadores Profissionais*
Treinadores que gostariam de realizar treinos secretos poderiam alugar o espaço para poder deixar seus jogadores mais a vontades sem se preocupar com espiões de times adversários ou da imprensa distraindo seus campeões

#### *Eventos Culturais*
Campos esportivos são excelentes para eventos culturais, seja shows, encontros de anime, eventos de tecnologias entre outros. Por caber uma quantidade enorme de pessoas e equipamentos é possível encontrar uma fonte de dinheiro nesta categoria, especialmente pela necessidade de reservar o dia inteiro para preparar o ambiente para receber os participantes

### **Design**
Aqui se encontra algumas explicações sobre as escolhas feitas para o visual do projeto

#### Cores
- Verde
Verde remete à grama, natureza, espaços abertos e atividade fisica

- Branco
Branco traz uma limpeza, algo corporativo e sério

A combinação das cores traz uma leveza, não agride a visão e ao mesmo tempo é uma
amostragem da essencia do projeto  

  ``É importante reforçar que dentro do futebol as cores são elementos importantes para as comunidades, por isso vale ressaltar que o projeto final vai permitir que os usuarios (pagantes) possam personalizar o site com suas cores de preferência.``

#### Formas
O site possui formas *geometricas pontudas*. Quadrados e retangulos são encontrados por todo o ambiente com uma unica exceção que será apontada mais à frente no texto.

Formas pontudas remetem à seriedade, corporativismo e à ideia de inflexibilidade. Com isso podemos atingir um público alvo mais sério. Apesar da seriedade ser quebrada na escolha da cor, acredito que o balanço da seriedade das pontas trás um equilibrio perfeito que é
capaz de atrair o mais sério dos CEO a até o mais amador dos jogadores, aquele que só
joga com uma latinha de cerveja na mão

##### Calendário
Como falado, apenas um local encontra formas que não são pontudas, e esse local está no calendario dentro da página agenda. O calendario possui **bordas arredondadas** para trazer dinamismo, com o objetivo de encorajar o cliente a finalizar a compra

##### Tabela de Formas
| Formas Pontudas | Formas Arredondadas |
| ----------- | ----------- |
| Seriedade, maturidade, corporativismo, responsabilidade, confiança, força | Diversão, imaturidade, juventude, exploração, suavidade, harmonia, acolhimento |

#### Tipogafia
O tipo da fonte usada é chamada de **Helvetica**, que é uma fonte sem serifa, de proporções simetricas de alta legibilidade. Ideal para ser facilmente lida por diversos tipos de pessoa. Além disso, traz um modernismo necessário para trazer o Instituto Acrion para o mundo digital
